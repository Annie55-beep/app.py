# 🚀 Flask Beginner Toolkit

## 📌 Overview
This project is a beginner-friendly toolkit for learning Flask, a lightweight Python web framework used to build APIs and web applications.

The project demonstrates how to set up Flask and build a simple API with multiple endpoints.

---

## 🎯 Objective
- Learn Flask using AI-assisted prompts  
- Build a simple REST API  
- Document the process for beginners  

---

## 🛠️ Tech Stack
- Python 3  
- Flask  
- VS Code
