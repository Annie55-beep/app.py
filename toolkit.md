# 📘 Flask Beginner Toolkit Documentation

## 1. Title & Objective
**Title:** Prompt-Powered Kickstart: Building a Beginner’s Toolkit for Flask  

**Technology Chosen:** Flask (Python web framework)  

**Why:**  
- Flask is lightweight and beginner-friendly.  
- Helps learn web APIs and routing quickly.  
- Allows building small, real projects fast.  

**End Goal:**  
Create a simple REST API with multiple endpoints and document the process for beginners using AI prompts.

---

## 2. Quick Summary of Flask
- **What is it?** Flask is a micro web framework for Python used to build web apps and APIs.  
- **Where is it used?** Backend web applications, REST APIs, lightweight microservices.  
- **Real-world example:** Many startups use Flask for small web services or prototypes due to its simplicity and flexibility.

---

## 3. System Requirements
- **OS:** Windows, Linux, Mac  
- **Tools/Editors:** VS Code  
- **Python version:** 3.x  
- **Packages:** Flask (`pip install flask`)
